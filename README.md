### O Poker Air é um aplicativo de Texas Hold'em multiplayer que utiliza uma TV, computador ou tablet como mesa host e até 9 jogadores conectados via smartphone, escaneando o QR Code exibido na tela principal.

# Como Funciona 🎯

* A **TV/Computador/Tablet** funciona como a mesa de jogo — exibe as cartas da mesa, fichas e estado da partida
* Cada **jogador escaneia o QR Code** exibido na tela host com seu smartphone
* O smartphone vira o **controle individual** do jogador — exibe suas cartas e permite realizar ações (call, raise, fold)
* Até **9 jogadores** simultâneos por mesa
* Acesse o jogo online [Aqui](https://lucasfujarra.github.io/Poker-Air/)

| Quadra | 🂡🂱🃁🃑 | 4 cartas do mesmo valor |
| Full House | 🂡🂱🃁🂮🂾 | Trinca + Par |
| Flush | 🂡🂩🂵🂳🂢 | 5 cartas do mesmo naipe |
| Sequência | 🂩🂸🂷🃆🃕 | 5 cartas em sequência de naipes diferentes |

# Funcionalidades 🛠

* ♠ Texas Hold'em completo com lógica de apostas
* 📺 Modo Host para TV, computador ou tablet
* 📱 Modo jogador via smartphone com QR Code
* 🔥 Tempo real via Firebase
* 🤖 Suporte a bots para completar mesas
* 🏠 Criação e entrada em salas
* 🎯 Avaliação automática de mãos
* 🔒 Sem instalação — roda direto no navegador
